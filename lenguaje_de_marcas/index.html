    • Origenes Lenguaje de Marcas

    Los lenguajes de marcas son una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación. El lenguaje de marcas más extendido es el HTML (HyperText Markup Language, lenguaje de marcado de hipertexto), fundamento del World Wide Web. Los lenguajes de marcas suelen confundirse con lenguajes de programación, pero no son lo mismo, ya que el lenguaje de marcado no tiene funciones aritméticas o variables, como poseen los lenguajes de programación. El marcado se usaba y se usa en la industria editorial y de la comunicación, así como entre autores, editores e impresores. Históricamente, el marcado se usaba para dirigir el proceso de presentación (impresión) mediante una linotipia o impresora programable. El padre de los lenguajes de marcas es Charles Goldfarb, investigador para la compañía IBM. Goldfarb participó en la creación del lenguaje GML, y posteriormente dirigió el comité que elaboró el estándar SGML, la piedra angular de los lenguajes de marcas.

    Diferentes estándares y organismos oficiales

    Los lenguajes de marcas tienen diferentes estándares y organismos oficiales que los regulan. Algunos de los estándares más comunes son HTML, XML y SGML. HTML es el lenguaje de marcas más utilizado y es el fundamento del World Wide Web. XML es un lenguaje de marcas que se utiliza para almacenar y transportar datos. SGML es un lenguaje de marcas que se utiliza para definir otros lenguajes de marcas. Los organismos oficiales que regulan los lenguajes de marcas incluyen la Organización Internacional de Normalización (ISO), la Comisión Electrotécnica Internacional (IEC) y la Unión Internacional de Telecomunicaciones (ITU). Estas organizaciones establecen estándares internacionales para los lenguajes de marcas y otros aspectos de la tecnología de la información.

    Orígenes y las diferencias entre CSS y SCSS.

    En cuanto a las diferencias entre CSS y SCSS, CSS es el lenguaje de estilo que cualquier navegador entiende para diseñar páginas web. SCSS es un tipo especial de archivo para SASS, un programa escrito en Ruby que ensambla hojas de estilo CSS para un navegador, y para información SASS agrega muchas funciones adicionales a las variables de tipo CSS, anidamiento y más, lo que hace que escribir CSS sea más fácil y más rápido.

    Principales etiquetas HTML5

    HTML5 es la última versión del lenguaje de marcado HTML. Las etiquetas HTML son elementos que se utilizan para definir el contenido y la estructura de una página web. Las etiquetas semánticas son un conjunto de etiquetas HTML que se utilizan para describir el contenido de una página web de una manera más significativa y estructurada. Algunas de las etiquetas semánticas más comunes en HTML5 son: header: Define una sección de encabezado para la página o una sección. nav: Define una sección de navegación para la página. article: Define una sección de contenido independiente y autónomo. section: Define una sección de contenido temático. aside: Define una sección de contenido relacionado pero no esencial. footer: Define una sección de pie de página para la página o una sección. Las etiquetas semánticas ayudan a los motores de búsqueda y otros agentes de usuario a comprender mejor el contenido de una página web. También pueden mejorar la accesibilidad y la usabilidad de una página web para los usuarios.

    Principales elementos CSS

    En CSS, los elementos son los bloques de construcción básicos que se utilizan para diseñar una página web. Algunos de los elementos más comunes son: Color: se utiliza para establecer el color de fondo o de primer plano de un elemento. Tipografía: se utiliza para establecer el tipo de letra, el tamaño y el estilo de un elemento. Espaciado: se utiliza para establecer el espacio entre los elementos. Bordes: se utiliza para establecer el ancho, el estilo y el color de los bordes de un elemento. Posicionamiento: se utiliza para establecer la posición de un elemento en la página. Fondo: se utiliza para establecer una imagen o un color de fondo para un elemento.
